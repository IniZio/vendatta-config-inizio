# GIT WORKFLOW

- Use feature branches: `feature/xxx`
- Commit messages MUST follow Conventional Commits: `feat:`, `fix:`, `docs:`, `chore:`, etc.
- Always use `git worktree` via Vendatta for isolation.
- NEVER push to `main` directly.
